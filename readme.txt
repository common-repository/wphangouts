== Description ==

This plugin allows you to embed a Google Hangout or Hangout on Air button directly on any post or page of your website.

== Installation ==

Upload in normal way, to your `/wp-content/plugins/` folder. 

1. Upload `wphangouts` folder to your `/wp-content/plugins/` directory  
2. Go to Settings > WPHangouts
3. Add your app ID (your Google+ ID - (numbers not vanity URL)
4. Choose your settings
5. When ready add the short code [wphangouts] where you want the button to appear.
